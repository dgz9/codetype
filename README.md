## Adding Snippets

Add new code snippets in `lib/snippets.ts`. Each snippet needs:
- `id`: Unique identifier
- `code`: The actual code to type
- `language`: javascript | typescript | python | rust | go
- `difficulty`: easy | medium | hard
- `name`: Display name
